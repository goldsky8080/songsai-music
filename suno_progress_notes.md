# SONGS Progress Notes

이 문서는 `D:\music` 프로젝트의 현재 구현 상태를 요약한 메모다.
이전 메모와 실험 기록을 정리해서, 지금 무엇이 완료됐고 무엇이 남았는지 빠르게 파악할 수 있게 정리한다.

## 1. 현재 완료된 기능

### 인증
- Google 로그인 전환 작업 진행
- 기존 이메일과 비밀번호 로그인 흐름은 축소 또는 정리 중

### 크레딧
- 무료와 유료 크레딧 분리
- 크레딧 부족 시 생성 차단
- 무료 쿠폰 충전 기능 추가

### 음악 생성
- SONGS API wrapper 연동
- 모델 선택 지원
- 보컬 선택 지원
- `직접 입력`, `AI가사`, `AI 자동 생성` 모드 지원

### 생성 결과
- 최근 생성 목록 표시
- 재생 기능
- 다운로드 API
- 듣기 활성화 후 90초 카운트다운 뒤 다운로드 허용
- 가사 txt 다운로드 fallback 보강
- 커버 이미지 기반 세로형 비디오 생성 지원
- 비디오 자막 실험 후 현재는 자막 없는 버전 유지

### 보너스곡 정책
- 실제 생성은 2곡 구조
- 기본은 1곡만 표시
- 추가 곡은 크레딧 차감 후 열기
- 7일 제한 정책 반영

### 관리자 기능
- 사용자 목록 조회
- 사용자 권한 변경
- 크레딧 수동 지급
- 무료 쿠폰 발급

## 2. SONGS API 관련 현재 상태

### 운영
- 운영 wrapper는 외부 상태와 CAPTCHA 흐름의 영향을 받는다
- 어떤 시점에는 잘 되고, 어떤 시점에는 세션이나 쿠키 문제로 흔들릴 수 있다

### 개발
- `dev-suno.songsai.org` 를 테스트용 wrapper로 사용 중
- 자동 가사 생성 경로는 wrapper 기준으로 계속 검증 중
- CAPTCHA와 세션 흐름은 아직 가장 민감한 영역이다

## 3. 지금 중요한 판단

- 메인 앱만의 문제가 아니라 wrapper와 외부 상태가 함께 영향을 준다
- dev와 운영은 같은 문제로 안 되는 것이 아니라, 막히는 지점이 서로 다를 수 있다
- 기능 개발과 wrapper 안정화는 분리해서 보는 것이 중요하다
