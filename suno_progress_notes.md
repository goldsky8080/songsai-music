# Suno Progress Notes

이 문서는 `d:\music` 프로젝트의 현재 구현 상태를 요약한 문서다.
예전 메모성 기록을 정리해서, 지금 무엇이 되었고 무엇이 남았는지를 빠르게 파악할 수 있게 다시 정리했다.

## 1. 현재 완료된 큰 기능

### 인증
- Google 로그인 전환 작업 진행 중
- 기존 이메일/비밀번호 로그인 라우트는 축소 또는 정리 중

### 크레딧
- 무료/유료 크레딧 분리
- 크레딧 부족 시 생성 차단
- 무료 쿠폰 충전 기능 추가

### 음악 생성
- Suno wrapper 연동
- 모델 선택 지원
- 보컬 선택 지원
- `직접 입력`, `AI가사`, `AI 자동 생성` 모드 지원

### 생성 결과
- 최근 생성 목록 표시
- 재생 기능
- 다운로드 API
- 듣기 가능 시점 이후 90초 카운트다운 후 다운로드 허용

### 보너스 트랙 정책
- 실제 생성은 2곡 구조
- 기본은 1곡만 표시
- 추가 곡은 크레딧 차감 후 열기
- 7일 제한 정책 반영

### 관리자 기능
- 사용자 목록 조회
- 사용자 권한 변경
- 크레딧 수동 지급
- 무료 쿠폰 발급

## 2. Suno 관련 현재 상태

### 운영
- 운영 wrapper는 외부 Suno 상태와 CAPTCHA 흐름에 영향을 받음
- 어떤 시점에는 잘 되고, 어떤 시점에는 세션/쿠키/CAPTCHA 이슈가 발생할 수 있음

### 개발
- `dev-suno.songsai.org` 를 테스트용 wrapper로 사용 중
- 자동 가사 생성 경로를 wrapper 중심으로 정리함
- CAPTCHA와 세션 흐름 때문에 완전한 안정화는 아직 진행 중

## 3. 지금 중요한 판단

- 메인 앱만의 문제가 아니라 wrapper와 Suno 외부 상태가 같이 영향을 준다
- dev와 운영은 같은 문제로 안 막히고, 막히는 단계가 다를 수 있다
- 따라서 기능 개발과 wrapper 안정화를 분리해서 보는 것이 중요하다

## 4. 지금 기준으로 남은 과제

- dev wrapper 반복 테스트
- 운영 반영 기준 정리
- Google 로그인 마무리
- 관리자 페이지 UX 개선
- 문서와 배포 절차 정리
